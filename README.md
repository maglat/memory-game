# 🎮 Memory Spiel

Ein modernes, visuell beeindruckendes Memory-Spiel mit animiertem Hintergrund und Feuerwerk-Effekten.

## ✨ Features

- **🎮 Einzelspieler-Modus**: Klassisches Memory für 1 Spieler
- **👥 Zweierspieler-Modus**: Wettbewerbsmodus für 2 Spieler gleichzeitig!
- **Animierter Hintergrund**: Sanfter Farbverlauf mit schwebenden Farb-Orbs
- **Feuerwerk-Effekte**: Bunte Explosionen bei jedem gefundenen Paar
- **3D-Karteneffekte**: Realistische Umklapp-Animationen
- **Timer & Statistiken**: Verfolge deine Zeit und Züge
- **Responsive Design**: Spielt auf allen Geräten (Desktop & Mobile)
- **Glowing-Effekte**: Der Titel pulsiert und leuchtet

## 🚀 Schnellstart

### Online spielen
Klicke hier: **https://maglat.github.io/memory-game/**

### Zweierspieler-Modus
1. Smartphone **vertikal** zwischen beiden Spielern platzieren
2. **Spieler 1** sieht nach oben, **Spieler 2** nach unten (oder umgekehrt)
3. Beide spielen gleichzeitig auf ihren eigenen Brettern
4. Wer zuerst alle 8 Paare findet, gewinnt! 🏆

### Lokal spielen
1. Klonen oder öffnen: `index.html` in deinem Browser
2. Wähle oben den Modus: **Einzelspieler** oder **Zweierspieler**
3. Klicke auf Karten, um sie aufzudecken

## 🎯 Spielregeln

### Einzelspieler
- Decke 2 Karten pro Zug auf
- Wenn sie passen: Sie bleiben offen und du erhältst ein Feuerwerk! 🎆
- Wenn sie nicht passen: Sie werden wieder umgedreht
- Finde alle 8 Paare so schnell wie möglich

### Zweierspieler
- Jeder Spieler hat sein eigenes 4x4 Spielfeld (8 Paare)
- Beide Spieler spielen **gleichzeitig** und **parallel**
- Jeder hat seinen eigenen Timer und Zähler
- Wer zuerst alle 8 Paare gefunden hat, **gewinnt**!
- **Format 9:16**: Das Smartphone wird vertikal zwischen die Spieler gelegt
- **Spieler 1**: Oben, normal ausgerichtet
- **Spieler 2**: Unten, um 180° gedreht (sieht alles richtig herum von seiner Seite aus)
